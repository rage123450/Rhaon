CTRScissorsRockPaper 클래스 작성
CTRScissorsRockPaper_GUI 클래스 작성

<구현이 달라지는 부분>
1. UI 컨트롤 갯수
- 각각 필요로 하는 컨트롤 갯수가 다르다.
- 사용하는 GUI도 다른게 있다.
 = 개인전 : 보상 분류 박스4개(이미지가 다름), NPC(이미지가 다름)
 = 단체전 : 보상 분류 박스10개(이미지가 다름), NPC(이미지가 다름),대기시간, 남은 인원, 
 	  가위바위보 선택한 사람을 표시하는 그래프, 변경 가능 횟수

2. 해당 단계에 따라서 NPC의 대사가 다른다.
ex1. 시작할 때...
개인전 : 좋아, 보상은 줄게. 허나 그냥은 심심하지... 게임 한번 어때?
단체전 : 다들 모여봐...후훗.

ex2. 게임시작할 때...
개인전 : 가위! 바위! 보! 할 줄 알지!?
단체전 : 자! 충분히 모인 것 같군... 시작해보자!

3. 보상 목록
개인전 : 기본 + 최대3개
단체전 : 최대10개

4. 게임 시작
개인전 : 건네주기 버튼 클릭후 바로 진행(정화의 돌10개 필요)
단체전 : 입장하고 대기시간 이후 진행

5. 게임 진행
개인전 : 
- 확인을 누르면 바로 응답, 
- 최대 3라운드
- 미조작 3회에 탈락(매번 2번까지는 가능)
- 무승부 판정 : 재게임
- 선택 : 3초 이내 자유롭게 변경 가능
단체전 : 
- 라운드 시작 5초 이후(그 동안은 선택 인원 갱신) 응답
- 최대 10라운드
- 미조작시 바로 탈락
- 매 라운드 현재 인원 갱신
- 무승부 판정 : 승리(원래 패배였다가 다시 변경됨.다시 변경될 수도 있음)
- 선택 : 1초에 한번씩 변경
